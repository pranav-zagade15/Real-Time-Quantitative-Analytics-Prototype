# Pre-Submission Checklist

- [ ] README is polished and contains demo + run instructions
- [ ] LICENSE (MIT) present
- [ ] requirements.txt pinned and present
- [ ] `tools/gen_synthetic_ticks.py` included and documented
- [ ] Unit tests present in `tests/` and deterministic
- [ ] CI workflow present and passing (GitHub Actions: `ci.yml`)
- [ ] Pre-commit configured (`.pre-commit-config.yaml`)
- [ ] No secrets in repository
- [ ] Screenshots or short demo clip attached to PR
- [ ] PR description includes: what changed, how to run demo, test status, known limitations

Copy into your PR body and tick off items when ready.
